# queerwinnipeg.ca

## Getting Started

1. First install [Bun](https://bun.sh/)

   ```powershell
   # Windows
   powershell -c "irm bun.sh/install.ps1 | iex"
   ```

   ```sh
   # Linux and macOS
   curl -fsSL https://bun.sh/install | bash
   ```

2. Run the following commands:

   ```sh
   bun install
   bun dev
   ```

3. Ta-da! You now have a working development instance.
